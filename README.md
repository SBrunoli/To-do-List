# 📝 To-Do List

<img width="793" height="441" alt="image" src="https://github.com/user-attachments/assets/8afc72f3-0988-4669-aba4-7702e9825f12" />


> Uma aplicação simples de lista de tarefas desenvolvida em **JavaScript**, **HTML** e **CSS**, com funcionalidades de criar, editar, excluir e filtrar tarefas. Tudo isso com persistência no **Local Storage** do navegador! 💾

---

## 🚀 Funcionalidades

| Emoji | Funcionalidade |
|-------|----------------|
| ➕ | Criar tarefas |
| ✏️ | Editar e atualizar tarefas |
| 🗑️ | Excluir tarefas |
| ✅ | Marcar tarefas como concluídas |
| 🔍 | Filtrar tarefas: **Todas**, **Concluídas**, **Pendentes** |
| 💾 | Persistência das tarefas usando Local Storage |

## 🎯 Objetivo do Projeto

Este projeto foi criado para treinar **JavaScript moderno**, manipulação do **DOM**, **Local Storage**, e boas práticas de UI/UX, criando um sistema funcional de tarefas que pode ser expandido futuramente.

---

## 🖥️ Como usar

1. Clone o repositório:

```bash
git clone git@github.com:SBrunoli/To-do-List.git
```

2. Abra o arquivo `index.html` no seu navegador.
3. Comece a adicionar suas tarefas! ✍️
